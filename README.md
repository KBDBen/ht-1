# 📊 CSV 데이터 AI 분석기

Google Sheets 스프레드시트를 AI가 자동으로 분석해주는 웹 서비스입니다.

## 이런 분들에게 유용합니다

- 스프레드시트 데이터를 빠르게 요약하고 싶을 때
- 데이터에서 패턴이나 이상값을 찾고 싶을 때
- 데이터 기반 액션 아이템을 도출하고 싶을 때

## 사용 방법

### 1단계: Google Sheets에서 CSV 링크 복사

1. Google Sheets에서 분석할 스프레드시트를 엽니다
2. **파일 → 공유 → 웹에 게시** 를 클릭합니다
3. 형식을 **CSV**로 선택하고 **게시** 버튼을 누릅니다
4. 생성된 링크를 복사합니다

### 2단계: 데이터 불러오기

1. 웹사이트에 접속합니다
2. 복사한 CSV 링크를 입력창에 붙여넣습니다
3. **데이터 불러오기** 버튼을 클릭합니다
4. 데이터가 테이블로 미리보기 됩니다

### 3단계: AI 분석 실행

1. **AI 분석 시작** 버튼을 클릭합니다
2. 잠시 기다리면 AI가 분석 결과를 카드 형태로 보여줍니다:
   - **전체 요약** - 데이터의 전반적인 내용
   - **발견된 패턴** - 데이터에서 발견된 규칙이나 경향
   - **이상값** - 눈여겨볼 특이한 데이터
   - **액션 아이템** - 데이터 기반 추천 행동

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프레임워크 | Next.js 14 (App Router) |
| 스타일링 | Tailwind CSS |
| AI 분석 | Claude API (Anthropic SDK) |
| 배포 | Vercel |
